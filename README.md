# PettisModel

Super simplified Michael Pettis economy model for self learning.

User should not focus on values presented on the graphs but rather on the changes that are happening when sliders are used. 

Good starting point is the tab "Advanced Theoretical Manual" where main teachings are presented and 5 core geopolitical scenarios are collected. These scenarios should be uses as baseline for understanding dynamics described by Michael Pettis.

Model was created via vibecoding with Gemini Pro. There is no guarantee that it represents well the teachings of Michel Pettis nor that any information included are correct. DYOR as always.
